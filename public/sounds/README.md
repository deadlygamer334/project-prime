# Focus Mixer Audio Files

This directory should contain the audio files for the Focus Mixer feature.

## Required Audio Files (21 Total)

- `pink-noise.mp3` - Balanced frequency noise
- `brown-noise.mp3` - Deep rumble noise

## Audio Requirements

- **Format**: MP3, OGG, or WAV
- **Loop**: Files should be seamless loops (no gaps)
- **Duration**: 30-60 seconds minimum for smooth looping
- **Quality**: 128-192 kbps is recommended for web
- **Volume**: Normalize all files to similar levels

## Testing

Once you've added the audio files, test the Focus Mixer by:
1. Navigating to `/ambience`
2. Using category filters to browse sounds
3. Clicking on sound cards to activate them
4. Adjusting volumes with vertical sliders
5. Hovering over cards to see focus tooltips
6. Testing presets for smooth fade transitions
7. Verifying the Randomize function
